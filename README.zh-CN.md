# AndroidClaw

[English README](README.md)

AndroidClaw 是一个以 **LLM 驱动手机操作** 为核心目标的 Android 原生智能体项目。它把大模型的任务规划能力与 Android 无障碍能力结合，让模型能够理解用户意图并执行真实手机操作。

## 项目灵感来源

本项目在设计思路上参考了多个优秀项目和产品，包括：

- OpenClaw
- NanoBot
- 豆包手机助手
- AutoGLM

项目目标不是复刻某一个实现，而是将其中可落地的思路沉淀成一个更适合 Android 端运行、扩展与部署的工程。

## 核心定位

让 **LLM 成为手机操作员**：

- 用自然语言理解用户需求
- 自动规划执行步骤并调用工具
- 读取当前界面信息并执行手势操作
- 以常驻守护进程方式保持可用性和稳定性

## 当前已支持功能

### LLM 与 Agent 能力

- 可配置模型提供方与模型参数
- Agent 会话管理与记忆
- 工具注册、路由与执行框架

### 手机操作能力（重点）

- 基于无障碍服务的 UI 交互
- 按文本点击 / 长按
- 按坐标点击 / 长按
- 滑动与滚动操作
- 全局按键：返回 / Home / 最近任务
- 按包名打开应用
- 读取当前屏幕文本与窗口标题

### 文件与命令工具

- 文件读写、列举、删除、建目录（应用范围）
- 通过工具层执行命令

### 守护与稳定性

- 前台守护服务
- 开机自启动
- WorkManager 健康检查与自愈
- 网络变化后的自动恢复流程

### 远程交互集成

- 飞书机器人接入
- 基于 WebSocket 的消息处理

### 应用侧体验

- 首次启动权限引导页
- 必需权限未满足时禁止进入主页
- 模型与飞书配置页面

## 技术栈

- Kotlin + Android SDK（minSdk 24, targetSdk 34）
- Room、WorkManager、Coroutines
- LangChain4j（模型与工具集成）
- OkHttp、Gson

## 构建方式

```bash
./gradlew assembleDebug
```

APK 输出路径：

- `app/build/outputs/apk/debug/app-debug.apk`
